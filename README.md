# :computer: Admin access template

Flask app with roles and permissions portal.

## :floppy_disk: Installation

```bash
python -m venv env
```

```bash
env/scripts/activate
```

```bash
python -m pip install --upgrade pip
```

```bash
pip install -r requirements.txt
```

## :runner: Run

```bash
flask run --debug
```
